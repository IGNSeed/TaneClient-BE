# TaneClient

## Build

Install the Desktop development with C++ workload for Visual Studio 2022 Community, then run:

```powershell
powershell -ExecutionPolicy Bypass -File scripts\build.ps1
```

The executable is generated at `build\dist\TaneClient.exe`.

## Opening the client

1. Run `build\dist\TaneClient.exe`.
2. Click the **Inject** button.
3. The official Minecraft for Windows application will open automatically.
4. Wait until the launcher displays **Injected**.
5. In Minecraft for Windows, press `Insert` to open or close the TaneClient menu.

For controller input, press `LB + RB + RStick` to open or close the TaneClient menu.

## Supported version

- Minecraft for Windows v26.21
